# 오늘의 매뉴를 골라주는 사이트
미리 입력해둔 식당들 중에 하나를 랜덤으로 선택해주는 사이트입니다.

## 프로젝트 배포 링크
<a href="https://main--today-s-menu.netlify.app/">https://main--today-s-menu.netlify.app/</a> <br/>
